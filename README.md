# Favorite Things Sampler Website

  ### By Taylor Somers

## Description

  A website that asks the user for input on their favorite item in five categories and then displays a subset of the favorite items the user submitted. This website showcases my new knowledge of arrays and their potentialities for collecting and organizing data input from users, a vital function in web-based programming.

## Installation

  1. Download a web browser, such as Apple Safari, Google Chrome, Microsoft Edge, Mozilla Firefox, or Yandex.
  2. Navigate to https://github.com/taylorsomers/favorite-things.
  3. Click the green "Clone or download" button at the right of the screen.
  4. Select "Download ZIP."
  5. Use a file extractor or unzip program (such as PeaZip, Unzipper, WinZip, Zipware, or 7-ZIP) to extract the ZIP files onto your computer.
  6. Open the directory containing the extracted files.
  7. Double-click the "index.html" file to run the program through your web browser.
